# HACKPAD
This project is a custom 6-key mechanical macropad designed for productivity and workflow automation.
#  Softwares Used
1. Autodesk Fusion
2. KiCad 9.0
3. Mu Editor

## Components Used
1) Microcontroller - **Seed Studio XIAO RP2040**
2) Switches(x6) - **Cherry MX Compatible Mechanical Switches**
3) RGB LEDs(x4) - **SK6812MINI (3535) Addressable RGB LEDs**
4) Case - 3D printed

## PCB Layout

**Key Pins:** - A0,A1,A2,A3,D4,D5
**RGB** - TX 
**POWER** - 5V VBUSs

## Keymapping
- Key 1 - Vol Up
- Key 2 - Vol Down
- Key 3 - Tap:Mute / Double Tap : Play-Pause
- Key 4 - Alt + Tab ( Tab Switch)
- Key 5 - Win + Ctrl + Left ( Previous desktop)
- Key 6 - Win + Ctrl + Right( Next desktop)
